# portfolio

[![CI](https://github.com/jiramos87/portfolio/actions/workflows/ci.yml/badge.svg)](https://github.com/jiramos87/portfolio/actions/workflows/ci.yml)

A full-stack developer showroom by Javier Ramos. Each exhibit shows two things: the shipped product, and exactly how it was built using agentic workflows (PRD to implement to verify to deploy).

**Live:** https://javierramos.dev

Status: building in public.

## What this is

Most portfolios show the product. This one also shows the build: the PRD that started each project, the commit and PR timeline, the metrics, and the agent-driven workflow behind it. The portfolio is itself the first exhibit, built with the same loop it documents.

## Stack

- Next.js (App Router) + TypeScript
- NestJS REST API
- PostgreSQL + Prisma
- Turborepo (pnpm) monorepo
- Tailwind CSS + shadcn/ui
- Vercel (web) + Railway (api + database)

## Built with

This project is built using my open-source agentic-dev-kit (Claude Code skills, an MCP server, and PRD templates). Link coming soon.
